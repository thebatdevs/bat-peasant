# `bat-peasant`

`bat-peasant` is a simple personalized coding agent system designed to help me build, maintain, and improve my TypeScript projects, maybe in some cases even in Python code too.

The goal of this project is to create a coding assistant that understands my preferred project structure, coding standards, testing patterns, error handling style, and development workflow. Instead of acting like a generic AI coding tool, `thebat-peasant` should behave like a project-aware engineering assistant that can generate code consistent with how I normally build backend systems.

## Purpose

The main purpose of `thebat-peasant` is to improve my development speed and consistency by giving an AI agent access to clear engineering rules, reusable project knowledge, and repository-specific context.

It should help with tasks such as:

- Implementing new features
- Fixing bugs
- Refactoring existing code
- Writing tests
- Generating documentation
- Reviewing code quality
- Following my TypeScript and Serverless coding standards
