# `theBatPeasant` milestone list

## Milestone 0: Define the goal clearly

Goal: Make sure this project is not “build an AI agent from scratch,” but “make existing coding agents behave like your personal trained engineer.”

```text
theBatPeasant = Codex/Copilot/Claude Code + repo memory + coding standards + workflows + verification gates
```

Done when

```text
You can explain the goal in 2-3 sentences.
```

## Milestone 1: Create the minimum AI context layer

Goal: Create the first usable version of the repo-native AI memory.

Done when:

Codex/Copilot can read these files and understand:

- what the project does
- how the code is structured
- how you write code
- how you write tests
- what commands must be run before final answer

## Milestone 2: Define mandatory agent workflows

Goal: Make the agent follow a predictable process instead of coding immediately.

Files to create:

```txt
ai/workflows/
├── feature.md
├── bug-fix.md
├── test-writing.md
└── code-review.md
```

Done when:

```txt
For any feature or bug fix, the agent must:
1. Read context
2. Analyze existing patterns
3. Create functional plan
4. Ask confirmation
5. Create coding-level plan
6. Ask confirmation
7. Implement
8. Create test plan
9. Ask confirmation
10. Test
11. Verify
12. Summarize results
```

## Milestone 3: Add reusable prompt templates

Goal: Stop writing long prompts manually every time.

Files to create:

```text
ai/prompts/
├── plan-feature.md
├── implement-feature.md
├── fix-bug.md
├── write-tests.md
├── review-diff.md
└── refactor.md
```

Done when:

```text
You can paste a short prompt like:
"Use ai/prompts/plan-feature.md. Add create user endpoint."

And the agent follows your expected process.
```
